# Documentation Index

이 디렉터리는 Confluence 이관과 내부 설계 검토를 위해 문서 성격별로 정리한다.

## Structure

| Directory | Purpose |
| --- | --- |
| `project/` | 프로젝트 진행 상태, 개발 컨텍스트, TODO, 결정 이력 |
| `operations/` | 기존 수동 운영 흐름, 실행 가이드, 전송/수집 절차 |
| `architecture/` | Kubernetes, shm, proc 접근 등 기술 구조 검토 |
| `decisions/` | 특정 설계 선택의 배경과 의사결정 기록 |

## Confluence Usage

- HTML 문서는 로컬 확인용으로 유지한다.
- Confluence에는 같은 이름의 Markdown 문서를 우선 사용한다.
- 그림은 Mermaid 코드블록으로 작성한다.
- Mermaid는 필요 시 draw.io로 변환해 Confluence에 첨부한다.
